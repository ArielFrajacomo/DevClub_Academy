Learning Experience

This project was much more of a hands-on experience with React and Tailwind.

Learning React was a good experience, and for this project I got to use Tailwind CSS for the first time. And, I was surprised by how alike it is to Bootstrap, it has an intuitive way to apply the styles and I had to write a lot less than just CSS, BUT Tailwind has so many little pieces that the ClassNames property can get out of control really fast, so it is best used when you have a good organization while writing the code.
