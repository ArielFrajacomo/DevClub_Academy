Installing React

Installing Globally

  1. Open Windows PowerShell
  2. Run the following command to install React globally: npm install --global yarn
    Only needed once, if you don't have Yarn installed

    Verify the installation by running: yarn --version

  3. Navigate to the directory where you want to create your React project: cd path/to/your/directory
  4. Create a new React project using the following command: npm create vite@latest
  5. Follow the prompts to set up your project:
    • Enter a name for your project (e.g., "my-react-app").
    • Select "React" as the framework.
    • Select "JavaScript" as the variant.
  6. Navigate into your project directory: cd my-react-app
  7. Start the development server: yarn dev
    or npm run dev

Installing Tailwind CSS

  1. Navigate to your React project directory if you haven't already: cd my-react-app
  2. Install Tailwind CSS and its dependencies using the following command: yarn add -D tailwindcss @tailwindcss/vite
    or npm install -D tailwindcss @tailwindcss/vite
  3. In the vite.config.js file, add the Tailwind CSS plugin: 
    import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'
import { defineConfig } from 'vite'              // <-- LINE to ADD

export default defineConfig({
  plugins: [
    react(), 
    tailwindcss()                               // <-- LINE to ADD
  ],
})

Utils

  • clsx: A utility for constructing className strings conditionally. npm install clsx tailwind-merge
    Add to src/lib/utils.js 
    import { clsx } from 'clsx';
import { twMerge } from 'tailwind-merge';

export function cn(...inputs) {
  return twMerge(clsx(inputs));
}

    how to use: 

    Without
    className={`px-4 py-2 ${isActive ? 'bg-black text-white' : 'bg-white'} ${disabled ? 'opacity-50 cursor-not-allowed' : ''}`} 

With
        className={cn(
    'px-4 py-2',
    isActive ? 'bg-black text-white' : 'bg-white',
    disabled && 'opacity-50 cursor-not-allowed'
    )}

  • react-icons: A popular library for including icons in React projects. npm install react-icons
  • Using @ as an alias for the src directory
    In your vite.config.js, add the following configuration to set up the alias:
    import { defineConfig } from 'vite';
import path from 'path';

export default defineConfig({
  resolve: {
    alias: {
      '@': path.resolve(__dirname, 'src'),
    },
  },
});

Axios

Axios is a promise-based HTTP client for making requests to APIs.

To install Axios, run the following command in your project directory:

npm install axios

React Router DOM

React Router DOM is a library for handling routing in React applications.

To install React Router DOM, run the following command in your project directory:

npm install react-router-dom

After installation, you need to create a Router component where you can import the necessary components from React Router DOM to set up routing in your React application.

import { BrowserRouter as Router, Route, Routes } from 'react-router-dom';

Then, you can define your routes using the <Routes> and <Route> components.

 
import { createBrowserRouter } from 'react-router-dom';

const router = createBrowserRouter([
    // Created in en-US and pt-BR versions for better accessibility (and SEO)
    {
        path: '/',
        element: <Home />
    },
    {
        path: '/page2',
        element: <Page2 />
     }
]);

Then, we need to wrap our application with the RouterProvider component and pass the router configuration to it.

import { RouterProvider } from 'react-router-dom';
import router from './routes.jsx';

createRoot(document.getElementById('root')).render(
  <StrictMode>
<RouterProvider router={router} />
  <StrictMode>
);
